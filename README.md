# Retail Sales Performance Analysis using SQL and Excel

## Objective
Analyse retail e-commerce data to generate key business insights related to revenue, products, customers, and country performance.

## Dataset
A sample of an e-commerce retail dataset containing transactional sales data, including invoices, products, quantities, prices, dates, and customer locations.

## SQL Queries
1. Total revenue from all sales  
2. Revenue by country  
3. Monthly sales trend  
4. Top 10 products by revenue  
5. Average order value

## Notes
This is a sample dataset for portfolio purposes. The queries demonstrate SQL aggregation, grouping, and analysis relevant for junior data analyst roles.
